# Simple codes file for creating, listing and deleting the S3 buckets.


Required Libraries:
```bash
boto3
os
sys
dotenv
```

Required Installations:
``` bash
boto3
dotenv
```

Add the following in your .env file:
```bash
AWS_ACCESS_KEY
AWS_SECRET_KEY
AWS_REGION
```

Tip: Pass the bucket name as the 1st argument.
